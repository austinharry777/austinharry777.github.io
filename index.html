Basic Tags

Tag Name Description
<!DOCTYPE> Document Type Declaration Defines the document type
<html> HTML document tag Container for all elements (except !DOCTYPE)
<head> Head tag Container for metadata
<title> Title tag Defines the title of the document (also tab name)
<body> Body tag Container for all visible elements
<h1> Heading tag Defines a heading (numbered 1 through 6)
<p> Paragraph tag Defines a paragraph
<br> Break tag Defines a line break
<hr> Horizontal Rule tag Defines a horizontal rule
<!-- --> Comment tag Defines a comment
<ul> Unordered List tag Defines an unordered list
<ol> Ordered List tag Defines an ordered list
<li> List Item tag Defines a list item
<img> Image tag Defines an image
<a> Anchor tag Defines a hyperlink
<div> Division tag Defines a division or section
<span> Span tag Inline container
<em> Emphasis tag Defines emphasized text
<strong> Strong tag Defines important text

Basic Tag Attributes

Attribute Name Description
id Identifier Defines a unique identifier for an element
class Class Defines a class name for an element
style Style Defines inline CSS for an element
title Title Defines extra information about an element
target Target Defines where to open the linked document
lang Language Defines the language of the element
dir Direction Defines the text direction
accesskey Access Key Defines a keyboard shortcut for an element
tabindex Tab Index Defines the tab order of an element
hidden Hidden Defines a hidden element
draggable Draggable Defines whether an element is draggable
contenteditable Content Editable Defines whether the content of an element is editable
alt Alternative Text Defines alternative text for an image

Semantic Tags

Tag Name Description
<header> Header tag Defines a header for a document or section
<nav> Navigation tag Defines a set of navigation links
<section> Section tag Defines a section in a document
<article> Article tag Defines an independent, self-contained article
<aside> Aside tag Defines content aside from the page content
<footer> Footer tag Defines a footer for a document or section
<figure> Figure tag Defines self-contained content
<figcaption> Figure Caption tag Defines a caption for a figure element
<mark> Mark tag Defines marked/highlighted text
<time> Time tag Defines a date/time
<video> Video tag Defines a video or movie
<audio> Audio tag Defines sound content
<source> Source tag Defines multiple media resources for media elements
<embed> Embed tag Defines a container for an external application

Table Tags

Tag Name Description
<table> Table tag Defines a table
<caption> Caption tag Defines a table caption
<colgroup> Column Group tag Specifies a group of one or more columns in a table for formatting
<col> Column tag Specifies column properties for each column within a colgroup element
<tbody> Table Body tag Groups the body content in a table
<thead> Table Header tag Groups the header content in a table
<tfoot> Table Footer tag Groups the footer content in a table
<tr> Table Row tag Defines a row in a table
<td> Table Data tag Defines a cell in a table
<th> Table Header tag Defines a header cell in a table

Table Attributes

Attribute Name Description
colspan Column Span Specifies the number of columns a cell should span
rowspan Row Span Specifies the number of rows a cell should span
headers Headers Specifies one or more header cells a cell is related to
scope Scope Specifies whether a header cell is a header for a column, row, or group of columns or rows
abbr Abbreviation Specifies an abbreviated version of a cell value
